# 📜 Cordel Moderno

Projeto desenvolvido como desafio do curso de **HTML5 e CSS3** do canal **Curso em Vídeo**, ministrado por Gustavo Guanabara.

## ✨ Sobre o projeto

O **Cordel Moderno** é uma página web que apresenta um poema em formato de cordel, combinando tipografia estilizada e imagens de fundo com efeito parallax, criando uma experiência visual mais imersiva e moderna.

O objetivo do desafio foi praticar conceitos de estruturação HTML e estilização com CSS, incluindo uso de fontes externas, imagens de fundo e responsividade.

## 🎯 Funcionalidades

- Página responsiva
- Efeito de fundo com parallax
- Tipografia personalizada com Google Fonts
- Estrutura semântica em HTML5
- Estilização moderna com CSS3

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- Google Fonts  

## 📚 Aprendizados

Durante o desenvolvimento, foram praticados conceitos como:

- Uso de `sections` e estrutura semântica
- Importação de fontes externas
- Background image com `background-attachment: fixed`
- Responsividade com `vw` e `vh`
- Estilização de textos e efeitos visuais
